# .aupExportTool
## .aupファイルのプラグイン, フォント, ファイル管理をちょっとだけ簡単にー
.aupExportToolでは、特定のプロジェクトファイルに使用されているフォントやスクリプトなどの依存関係を一覧として出力するだけでなく、使用されている画像や動画ファイルを.zip形式の単一アーカイブファイル化することができます。   
  
## 動作環境
✅ windows7以上のOS  
✅ 1GHz以上のプロセッサ  
✅ 512MB以上のRAM  
✅ .Net SDK6.0(環境に応じてx86またはx86_64)
